# LinguaSRS Bot

Telegram spaced-repetition flashcard bot for language learners: create/import decks, review with four-button ratings, set daily new-card limits, and get reminders.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
